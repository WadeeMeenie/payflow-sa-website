# Meta Pixel Operations

## Advanced Matching

The website does not currently collect lead form fields, so Advanced Matching is not implemented in code. Enable Automatic Advanced Matching in Meta Events Manager if appropriate. Never send employee names, ID numbers, salary or wage data, PAYE, UIF, ETI, reports, or other payroll data to Meta.

## Domain Verification

Use DNS TXT verification for `mypayflowsa.co.za` because it verifies the root domain independently of the GitHub Pages/Vite deployment.

Get the TXT record from Meta Business Suite > Brand Safety > Domains > Add Domain > DNS Verification, then add it at your domain DNS provider.
